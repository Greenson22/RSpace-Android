# 🚀 RSpace Next - Your Personal Learning & Productivity Hub

Selamat datang di **RSpace Next**, sebuah aplikasi *Personal Knowledge Management (PKM)* yang dirancang untuk membantu Anda mengatur materi pelajaran, melacak diskusi, dan mengoptimalkan proses belajar Anda.

**RSpace Next** adalah aplikasi *offline-first* yang memberi Anda kendali penuh atas data Anda[cite: 1]. Aplikasi ini difokuskan pada manajemen konten lokal, pengeditan berkas, serta kemudahan transfer data antar perangkat tanpa memerlukan akses internet[cite: 1].

---

## ✨ Fitur Utama

### 🧠 Manajemen Konten Hirarkis
Atur pengetahuan Anda secara terstruktur menggunakan hierarki **Topik > Subjek > Diskusi > Poin**[cite: 1]. Setiap diskusi dan poin dapat diberikan kode repetisi (*Repetition Code*) untuk menjadwalkan kapan Anda harus mengulang/meninjau materi tersebut[cite: 1].

### 📚 PerpusKu (*Perpustakaan Saya*)
Kelola semua file materi referensi Anda di satu tempat yang terorganisir[cite: 1]. Mendukung penelusuran file dan pembacaan langsung di dalam aplikasi[cite: 1].

### 📝 Editor Konten Bawaan (HTML & Markdown)
Aplikasi ini dilengkapi dengan editor teks bawaan untuk file HTML dan Markdown (`.md`) yang mendukung berbagai tema *syntax highlighting* (seperti Atom One Dark, GitHub, Android Studio, dll.)[cite: 1]. 

### 🌐 WebView & PDF Viewer Terintegrasi
Buka tautan web eksternal dan lokal secara internal menggunakan WebView bawaan yang mendukung fitur Bookmark[cite: 1]. Tersedia juga PDF Viewer khusus yang mendukung konversi HTML ke PDF untuk platform desktop (Linux) menggunakan pustaka bawaan sistem (`weasyprint` atau `wkhtmltopdf`)[cite: 1].

### ☁️ Data Center (Backup & Local Sharing)
Pusat kendali data Anda. Fitur ini memungkinkan Anda:
- **Backup Lokal:** Mengompresi seluruh folder utama (`RSpace_data` dan `PerpusKu`) menjadi file ZIP untuk keamanan data[cite: 1].
- **Local Sharing (Sinkronisasi Nirkabel):** Mengirim dan menerima data antar perangkat secara *wireless* melalui jaringan Wi-Fi lokal yang sama tanpa memerlukan koneksi internet, menggunakan sistem komunikasi dua arah (Server/Client)[cite: 1].

## 🛠️ Cara Instalasi

Karena aplikasi ini **tidak didistribusikan melalui Google Play Store**, Anda perlu menginstalnya secara manual:

1. Unduh file `app-release-v2.0.0.apk` dari tautan di atas.  
2. Buka file yang telah diunduh di perangkat Android Anda (biasanya ada di folder **Downloads**).  
3. Perangkat Anda mungkin akan meminta izin untuk *Install from unknown sources* (**Instal dari sumber tidak dikenal**). Izinkan permintaan ini untuk melanjutkan.  
4. Ikuti petunjuk di layar untuk menyelesaikan instalasi.  
5. Buka aplikasi **RSpace Next** dan mulailah mengatur data Anda!
